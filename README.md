# ⚔️ Endfield: Duel Arena

Интерактивная платформа для проведения турниров и дуэлей в мире **Endfield**. Приложение позволяет пользователям собирать коллекции персонажей и оружия, вызывать соперников на дуэль, проходить фазы драфта (бан/пик) и сражаться с боссом на время.

# Сайт доступен по ссылке:
https://endfield-smoky.vercel.app/

## 🌟 Основные возможности

### 👤 Профиль пользователя
- **Коллекция**: Добавление и управление личными персонажами и оружием.
- **Констелляции (Пробуждение)**: Уникальная система улучшения предметов. Каждый пользователь может устанавливать уровень консты (C0–C6) для своих копий персонажей и оружия независимо от глобальной базы.
- **Редактирование**: Удобный интерфейс для изменения параметров коллекции через модальные окна.

### ⚔️ Система Дуэлей
- **Создание вызова**: Отправка приглашений другим игрокам в реальном времени.
- **Фаза Драфта**:
  - Поочередные баны и пики персонажей.
  - Таймер хода с системой дополнительного времени.
  - Административный контроль (отмена ходов, редактирование выбора).
  - Отображение констелляций выбранных персонажей.
- **Выбор оружия**:
  - Выбор 4 единиц оружия из личной коллекции после завершения драфта героев.
  - Фильтрация доступного оружия только из инвентаря игрока.
- **Битва с Боссом**:
  - Фаза прохождения контента на время.
  - Система ретраев (попыток) с лимитами.
  - Автоматическое определение победителя по лучшему времени.
  - История результатов и административное редактирование итогов.

### 👁️ Режим зрителя и Архив
- Просмотр любых завершенных дуэлей в режиме "только чтение".
- Навигация между фазами драфта и боя в архивных записях.
- Отображение полных составов команд и уровней пробуждения (конст) участников.

### 🛡️ Безопасность и Архитектура
- **Защита данных**: Использование Firebase Security Rules для разграничения прав доступа. Пользователи могут редактировать только свои данные и результаты в своих дуэлях.
- **Конфиденциальность**: Конфигурация Firebase вынесена в переменные окружения (`.env.local`) и не попадает в репозиторий.
- **Модульность**: Компонентная архитектура (Draft, Weapons, Results) с разделением логики и стилей.
- **State Management**: Глобальное состояние на Redux Toolkit с асинхронными thunk'ами для работы с Firebase.

## 🛠 Технологический стек

- **Framework**: [Next.js 14+](https://nextjs.org/) (App Router)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **State Management**: [Redux Toolkit](https://redux-toolkit.js.org/)
- **Backend / DB**: [Firebase](https://firebase.google.com/) (Realtime Database, Auth)
- **Styling**: SCSS Modules / Global SCSS
- **Notifications**: [React Hot Toast](https://react-hot-toast.com/)
- **Icons**: Emoji & Custom SVG

## 📂 Структура проекта

```
src/
├── app/                  # Страницы приложения (Next.js App Router)
│   ├── duel/[duelId]/    # Страница конкретной дуэли
│   ├── user/             # Профиль пользователя и коллекция
│   ├── admin/            # Админ-панель
│   └── layout.tsx        # Основной лейаут с провайдерами
├── components/           # Переиспользуемые компоненты
│   ├── duels/            # Компоненты дуэлей (Draft, Weapons, Results)
│   ├── Modal/            # Модальные окна (редактирование, выбор)
│   └── ItemCard.tsx      # Карточка предмета с поддержкой конст
├── lib/                  # Утилиты и конфигурация
│   ├── firebase.ts       # Инициализация Firebase
│   ├── store.ts          # Настройка Redux Store
│   ├── slices/           # Redux слайсы (auth, duels, users, items)
│   ├── config/           # Константы и настройки игры
│   └── types/            # TypeScript интерфейсы
└── styles/               # Глобальные стили и миксины SCSS
```
